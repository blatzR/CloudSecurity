# Summary

In AWS, a **user** is a person or a computer that can do things on the cloud - just like you right now!

We'll be using, AWS **Identity and Access Management** (IAM) service to control who is authenticated (signed in) and authorized (has permissions) in your AWS account.

We'll launch an EC2 instance, then control who has access to it by creating some IAM policies and user groups.
